Вопрос 1
Изначально i = 5.

Условие i > 0 — истинно, заходим в цикл.

i *= 3; → i = 5 * 3 = 15

i *= -1; → i = 15 * (-1) = -15

Возвращаемся к проверке условия: i > 0 → -15 > 0 → ложь.

Цикл прекращается после первой итерации.

Ответ:
Цикл выполнится 1 раз, потому что после первой итерации i становится отрицательным, и условие i > 0 больше не выполняется.

Вопрос 2
Цикл for идёт с шагом 2 от 1 до 99 → 50 итераций.
Команда j = j - 1; выполняется один раз за итерацию.

Ответ: строка j = j - 1; выполнится 50 раз.
